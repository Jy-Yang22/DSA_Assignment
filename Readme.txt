1. Run the MemberMaintainance.java. It will show a main menu.
2. Select 1 will go to add member.
3. Select 2 will go to delete member.
4. Select 3 will go to search member.
5. Select 4 will go to modify member.
6. Select 5 will go to Generate report.
7. Select 6 will exit the program.

add()
user need to follow the message to insert the input.

delete()
user need to follow the message to insert the input. Ensure the input insert must exists inside the list.

search()
user need to follow the message to insert the input. Ensure the input insert must exists inside the list.

modify()
user need to follow the message to insert the input. Ensure the input insert must exists inside the list.
